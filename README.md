# N-Queens Problem ♛

Python implementation of the N-Queens problem using backtracking algorithm.

## Rules
- No two queens can be in the same row
- No two queens can be in the same column
- No two queens can be on the same diagonal

## Algorithm
- Backtracking
- Recursion
- Constraint solving

## Run
python n_queens.py

## Input
Enter value of N (e.g., 4, 8, 10)
